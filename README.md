# Ordle

A complete Wordle implementation distributed as a single, self-contained HTML file.

Ordle was built as a technical challenge: deliver a fully featured word game: filtered dictionary, letter logic, and in-game word definitions in under 0.2 MB, with no dependencies.

---

## Features

- **Curated word list**: only real, common English words; no random letter strings
- **Word definitions**: look up any word in the `INFOS` tab after each round
- **Smart letter hints**: letters are greyed out based on confirmed positions to reduce guesswork
- **Repeated letter handling**: unresolved repeated letters are marked with a distinct color
- **Offline support**: runs entirely in the browser from a local file
- **Cross-platform**: compatible with Chrome, Firefox, and mobile browsers
- **No account required**: free to use, no ads, no tracking
- **Open source**

---

## Getting Started

### Play Online

Visit [vovencio.github.io/Ordle](https://vovencio.github.io/Ordle/): no installation required.

### Play Offline

1. Download `ordle.html` from the [Releases](https://github.com/Vovencio/Ordle/releases) tab.
2. Open the file in any modern browser.

### Mobile

Tap the **Go Fullscreen** button below the game board for the intended mobile experience.
